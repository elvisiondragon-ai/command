# eL vision Command Center

Command and Control for eL vision Group.

## Domain
[command.elvisiongroup.com](https://command.elvisiongroup.com)

## Local Development
1. Clone the repository
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
4. Build for production: `npm run build`

## Features
- AI Agent Monitoring
- Content Pipeline Management
- Ads Performance Hub
- Social Performance Tracking
- Research & Insights Engine
